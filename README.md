# MERN E-Commerce
## BackEnd
1. Add package.json
    - `npm init`
2. Add Folders for 
    - `config`
    - `controller`
    - `middlewares`
    - `models`
    - `routes`
        - `index.js`
3. Install `npm install express mongoose bcrypt body-parser dotenv`
    - `express`
    - `mongoose`
    - `bcrypt`
    - `body-parser`
    - `dotenv`
